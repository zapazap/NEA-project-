To set up:<br/>
  >have python installed<br/>
  >run these commands in terminal:<br/>
      >pip install flask<br/>
      >pip install sqlalchemy<br/>
      >pip install flask.login<br/>
  >create a virtual environment using F1 + select interpreter:<br/>
      >in interpreter set up virtual environment<br/>
      >choose any python version 3.1+<br/>
      >add interpreter paths:<br/>
         >flask<br/>
         >sqlalchemy<br/>
         >flask.login
